# skills-repo

## soul-file skill
A standard, heavily logic-driven Node.js file (like a controller for a distributed JSON mapper) might be 1,000 lines long. That easily translates to 8,000–12,000 tokens of raw syntax, loops, error handling, and boilerplate.

If an agent needs to know what that file does or what it exports, dumping the raw file into the context window burns 10k tokens instantly.

A Soul File for that exact same module will be about 30–50 lines of highly dense, semantic markdown. That’s roughly 300 to 500 tokens. You are effectively compressing the file's architectural footprint by 95%.
